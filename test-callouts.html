GitHub-style Callouts Test

Note Callout

Note: This is a note callout. Use this to provide helpful context or additional information.

Multiple paragraphs are supported within callouts.

Tip Callout

Tip: This is a tip callout. Use this to provide helpful advice or best practices.

Important Callout

Important: This is an important callout. Use this to highlight crucial information.

Warning Callout

Warning: This is a warning callout. Use this to alert users about potential issues.

Caution Callout

Caution: This is a caution callout. Use this to warn about serious risks or dangers.

Markdown Syntax

In markdown, you would write these as:

[!NOTE]
This is a note callout.

[!TIP]
This is a tip callout.

[!IMPORTANT]
This is an important callout.

[!WARNING]
This is a warning callout.

[!CAUTION]
This is a caution callout.
